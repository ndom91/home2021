## 🏗 Quick Start

1. Clone repository

```sh
git clone https://github.com/ndom91/home2021
```

2. Install dependencies

```sh
cd home2021 && pnpm install
```

3. Start dev server

```sh
pnpm dev
```

The site should be up and running at [localhost:3003](http://localhost:3003).
